Sobre o Projeto

Este projeto é um CRUD (Create, Read, Update, Delete) de produtos, desenvolvido utilizando uma combinação de tecnologias modernas e robustas. O objetivo é fornecer uma interface fácil de usar para a gestão de produtos, permitindo que os usuários realizem operações básicas como adicionar, visualizar, atualizar e excluir produtos em um banco de dados MySQL.

Tecnologias Utilizadas

  • Java JSP: Utilizado para criar páginas dinâmicas e interativas, permitindo a integração com o banco de dados e a execução de operações CRUD.
  • HTML: Utilizado para estruturar o conteúdo das páginas, garantindo uma apresentação clara e organizada das informações.
  • CSS: Aplicado para estilizar a interface do usuário, proporcionando uma experiência visual agradável e responsiva.
  • MySQL: Banco de dados relacional usado para armazenar e gerenciar os dados dos produtos, garantindo persistência e integridade das informações.

Funcionalidades

  • Criar Produtos: Adicione novos produtos ao banco de dados com detalhes como código, nome, marca e preço.
  • Visualizar Produtos: Consulte a lista de produtos armazenados e visualize suas informações.
  • Atualizar Produtos: Modifique as informações de produtos existentes conforme necessário.
  • Excluir Produtos: Remova produtos do banco de dados que não são mais necessários.

Como Começar

Para iniciar o uso do CRUD de produtos, clone o repositório do projeto e configure o ambiente conforme as instruções no arquivo README. Certifique-se de ter o MySQL configurado e um servidor de aplicações Java como o Apache Tomcat para executar as páginas JSP.
